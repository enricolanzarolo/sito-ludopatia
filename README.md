# Sito Ludopatia

Sito web realizzato in seconda superiore (2023) come progetto scolastico.

Il sito tratta il tema della ludopatia con pagine informative, storia, campagne di sensibilizzazione e un mini gioco di blackjack implementato in JavaScript.

## Tecnologie
HTML, CSS, JavaScript

## Note
Primo approccio allo sviluppo web. Codice monolitico senza suddivisione ottimale dei file — progetto di apprendimento.
